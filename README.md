El uso desmedido de tiktok; como afecta en nuestro cerebro.
Esta plataforma usa un algoritmo sumamente adictivo, el cual puede ser perjudicial para la salud mental.
Se presenta un deterioro cognitivo elevado, se disminuye la retención de atención sostenida dado a que el cerebro se acostumbra a tener dopamina sin medida y eso resulta en que nuestro cerebro ya no busque una creación de está de forma "natural".
El uso desmedido hace que no se pueda mantener atención por mas de seis minutos, lo cual resulta problemático cuando se requieren hacer tareas largas o con atención sostenida.
